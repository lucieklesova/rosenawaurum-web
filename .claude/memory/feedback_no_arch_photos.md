Don't use arch, blob, or other non-standard shapes for photo frames. The user prefers standard rounded rectangle (`border-radius: 1.25rem`) frames.

**Why:** User found the arch shape visually unappealing compared to standard rounded corners.
**How to apply:** When styling image containers on this project, stick to simple `border-radius` values. Don't experiment with complex `border-radius` multi-value shapes or `clip-path` on photos.
